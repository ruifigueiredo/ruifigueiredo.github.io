Disseminação de Software de Código Alberto Caso ITIJ

2011-01-15
#artigo  #talk 

O Instituto das Tecnologias de Informação na Justiça (ITIJ) é o instituto credenciado pelo Ministério da Justiça para a gestão do parque informático, rede de comunicações e evolução do sistema de informação no Estado Português do âmbito da Justiça. [1] Analisaremos ao longo do trabalho a implementação de Open Source Software [OSS] neste instituto durante o triénio de 2006-2008. Para tal, estudaremos, com os modelos próprios, o planeamento para o triénio, tal como as aplicações realmente implementadas até 2008. Por fim e, após as conclusões tiradas comparando os dois casos anteriores, faremos sugestões para o futuro do instituto.
